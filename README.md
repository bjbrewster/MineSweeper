# MineSweeper

Simple console based MineSweeper mine field generator and solver.

Built with Visual Studio 2019 Community.

## Usage

```
MineSweeper <WIDTH> <HEIGHT> <COUNT>
```

| Param | Description |
|---|---|
| WIDTH | width of the mine field from 1 to 25 |
| HEIGHT | height of the mine field from 1 to 25 |
| COUNT | number of mines on the field from 1 to number of cells |
